
1. 목적과 범위
지역별 GRDP(지역내총생산)에 따른 의료 인프라와 실제 의료 수요의 차이를 데이터로 밝히려는 과제의 실행 프롬포트
목표는 2개 이상의 공공 데이터 소스를 서로 연결해 SQL 기반의 분석, 파이썬 데이터 처리, 시각화, 발표 자료를 통해 실용적인 인사이트를 얻는 것이며, 모든 과정은 재현 가능하도록 남깁니다.
2. 주제와 연구질문
주제: 지역별 GRDP에 따른 의료 인프라와 의료 수요의 격차 분석
연구질문:
GRDP가 높은 지역에서 의료 인프라가 충분한가?
실제 의료 이용은 GRDP와 어떤 관계를 보이며, 특정 지역에서 의료 과부하가 발생하는가?
인구 구조 변화가 의료 수요에 어떤 영향을 미치는가?
알파 가설: 고령화 및 인구 구조 변화가 저소득 지역의 의료 수요를 더 증가시키는가?
데이터 소스와 결합 방식
데이터 A: GRDP 지표(시도별/시군구별, 연도 포함)
데이터 B: 의료 이용 통계(진료현황, 외래/입원, 내원일수 등)
데이터
공통 키: 지역명(시도/시군구) 또는 지역 코드
결합 방법: 데이터 A와 B를 JOIN
분석 방법과 도구
도구: Python(Pandas) + SQL(SQLite) + 시각화 도구(Streamlit 대시보드 가능)
핵심 분석: JOIN, GROUP BY, SUM/COUNT, 환자1인당 연간 내원일수 같은 지표 파생
시각화 아이디어: 막대그래프, 산점도
